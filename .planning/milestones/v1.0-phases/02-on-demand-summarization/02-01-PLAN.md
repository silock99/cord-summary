    **src/bot/formatting/__init__.py** -- Create package init:
    ```python

    ```

    **src/bot/formatting/embeds.py** -- Create embed builder module with these components:

    Constants:
    - `EMBED_DESC_LIMIT = 4096`
    - `EMBED_COLOR = 0x5865F2` (Discord blurple)

    Function `build_summary_embeds(summary_text: str, channel_name: str, timerange_label: str) -> list[discord.Embed]`:
    - If summary_text is empty or whitespace, return a single embed with description "No summary content generated."
    - Call `_split_into_topics(summary_text)` to get sections
    - Accumulate sections into chunks that fit under EMBED_DESC_LIMIT
    - When adding a section would exceed the limit, finalize current chunk as an embed and start new chunk
    - If a single section exceeds EMBED_DESC_LIMIT, truncate to `EMBED_DESC_LIMIT - 20` chars and append `"\n*(...truncated)*"`
    - Join sections with `"\n\n"` separator
    - Return list of embeds

    Function `_split_into_topics(text: str) -> list[str]`:
    - Use `re.split(r'(?=\n\*\*)', text)` to split before lines starting with `**`
    - Strip each part, filter empty strings
    - Return list of topic sections

    Function `_make_embed(description: str, channel_name: str, timerange_label: str, index: int) -> discord.Embed`:
    - Create `discord.Embed(description=description, color=EMBED_COLOR)`
    - Title: `f"Summary: #{channel_name}"` if index == 0, else `f"Summary: #{channel_name} (continued)"`
    - Footer: `f"Period: {timerange_label}"`
    - Return embed

    **tests/test_embeds.py** -- Create tests:
    1. `test_single_short_summary` -- text under 4096 chars returns 1 embed with correct title, color, footer
    2. `test_empty_summary` -- empty string returns 1 embed with "No summary content generated."
    3. `test_multi_topic_split` -- construct text with 3 bold-header topics where total > 4096 chars, verify multiple embeds, second has "(continued)" in title
    4. `test_single_oversized_section` -- one section > 4096 chars gets truncated with "(...truncated)"
    5. `test_topic_splitting_regex` -- verify _split_into_topics correctly splits on bold headers

    Tests should import `from bot.formatting.embeds import build_summary_embeds, _split_into_topics`.
