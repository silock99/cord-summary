    ```python
    TIMERANGE_CHOICES = [

        app_commands.Choice(name="Last 4 hours", value=240),
        app_commands.Choice(name="Last 12 hours", value=720),
        app_commands.Choice(name="Last 24 hours", value=1440),
    ]

    TIMERANGE_LABELS = {30: "Last 30 minutes", 60: "Last 1 hour", 240: "Last 4 hours", 720: "Last 12 hours", 1440: "Last 24 hours"}
    ```

    Function `register_summary_command(bot)` that defines and registers the command:

    1. Decorator: `@bot.tree.command(name="summary", description="Summarize recent channel activity")`
    2. `@app_commands.describe(timerange="Time period to summarize", channel="Channel to summarize (defaults to current channel)")`
    3. `@app_commands.choices(timerange=TIMERANGE_CHOICES)`
    4. Signature: `async def summary(interaction: discord.Interaction, timerange: int = 240, channel: discord.TextChannel | None = None)`

    Command handler logic (in order):

    **Step 1 - Defer immediately (per D-04):**
    ```python
    await interaction.response.defer(ephemeral=True)
    ```

    **Step 2 - Resolve target channel (per D-09):**
    ```python
    target = channel or interaction.channel
    ```

    **Step 3 - Validate allowlist (per D-08, D-10):**
    Check `target.id not in bot.settings.allowed_channel_ids`. If not allowed:
    ```python
    mentions = ", ".join(f"<#{cid}>" for cid in bot.settings.allowed_channel_ids)
    await interaction.edit_original_response(
        content=f"This channel isn't enabled for summaries. Available channels: {mentions}"
    )
    return
    ```
    Also handle empty allowlist: if `not bot.settings.allowed_channel_ids`, respond with `"No channels are configured for summaries. Ask an admin to set ALLOWED_CHANNEL_IDS."` and return.

    **Step 4 - Calculate time range:**
    ```python
    now = datetime.now(timezone.utc)
    after = now - timedelta(minutes=timerange)
    timerange_label = TIMERANGE_LABELS.get(timerange, f"Last {timerange} minutes")
    ```

    **Step 5 - Fetch and preprocess messages:**
    Call `fetch_messages(target, after)` then `preprocess_message(msg, interaction.guild)` for each raw message, filtering None results. Use the same pattern as `summarize_channel` but we need the message count for quiet threshold check.

    **Step 6 - Check quiet threshold (per D-11, D-12):**
    ```python
    if len(processed) < bot.settings.quiet_threshold:
        await interaction.edit_original_response(
            content=f"No significant activity in {target.mention} in the {timerange_label.lower()}."
        )
        return
    ```

    **Step 7 - Summarize via provider:**
    ```python
    try:
        summary_text = await summarize_messages(
            bot.provider, processed, bot.settings.max_context_tokens
        )
    except SummaryError as e:
        await interaction.edit_original_response(content=f"Summary failed: {e}")
        return
    ```

    **Step 8 - Build embeds and send (per D-06):**
    ```python
    embeds = build_summary_embeds(summary_text, target.name, timerange_label)
    await interaction.edit_original_response(embed=embeds[0])
    for extra in embeds[1:]:
        await interaction.followup.send(embed=extra, ephemeral=True)
    ```

    Log the command usage: `logger.info(f"/summary by {interaction.user} in #{target.name} ({timerange_label}): {len(processed)} messages -> {len(embeds)} embed(s)")`
