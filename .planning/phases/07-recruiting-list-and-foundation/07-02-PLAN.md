    **`/recruit-add`** (per D-04, D-06, D-07, RECRUIT-01):

    - `@app_commands.describe(name="Player name to remove")`
    - Defer ephemeral, resolve sport, call `bot.recruit_store.remove_player(sport, name)`
    - If player returned: "Removed **{name}** from the {sport} recruiting list."

    - Defer (NOT ephemeral -- public response per D-13)
    - Resolve sport, get players via `bot.recruit_store.list_players(sport)` (already sorted by position)
    - If no players: send single embed with description "No players on the {sport} recruiting list yet."
    - Build embed(s) using field-per-player layout per D-01:

      - Color: `0x0051BA` (KU blue)
      - For each player, add an embed field:
        - `name="{player.name} {'star_emoji' * player.stars if player.stars else 'Unrated'}"` (use "\u2b50" for star emoji)
        - `value="{player.position} | {player.school}\nAdded {discord.utils.format_dt(datetime.fromisoformat(player.added_at), style='R')}"` (per RECRUIT-04)
        - `inline=False`
      - Split into multiple embeds at 25 fields (Discord hard limit per research Pitfall 1)
      - Set footer on last embed: "Last updated: {datetime.now(timezone.utc).strftime('%b %d, %Y %I:%M %p UTC')}"
      - Truncate field name to 256 chars (per research Pitfall 3)
    - Send all embeds via `interaction.edit_original_response(embeds=embeds)` (for first batch) or `interaction.followup.send(embeds=...)` if >10 embeds

    ```python
    @recruit_add.error
    @recruit_remove.error

    async def recruit_error(interaction: discord.Interaction, error: app_commands.AppCommandError):
        if isinstance(error, app_commands.CheckFailure):
            if interaction.response.is_done():
                await interaction.followup.send(str(error), ephemeral=True)
            else:
                await interaction.response.send_message(str(error), ephemeral=True)
        else:
            logger.error(f"Recruiting command error: {error}", exc_info=error)
            msg = "Something went wrong. Please try again later."
            if interaction.response.is_done():
                await interaction.followup.send(msg, ephemeral=True)
            else:
                await interaction.response.send_message(msg, ephemeral=True)
    ```
