async def send_error_alerts(bot, label: str, errors: list[str]) -> None

Implementation details:
- Module-level: `import logging`, `import discord`, `from datetime import datetime, timezone`
- Constant: `ERROR_EMBED_COLOR = 0xED4245` (Discord red)
- Read admin IDs from `bot.settings.admin_user_ids`
- If empty list: log warning `f"No admin users configured, skipping error DM for: {label}"` and return (per D-16)
- Build description as numbered list: `"\n".join(f"{i+1}. {e}" for i, e in enumerate(errors))`
- If description exceeds 4096 chars, truncate to 4080 chars and append `"\n...(truncated)"`
- Create `discord.Embed` with title=`f"Summary Errors: {label}"`, description=description, color=ERROR_EMBED_COLOR, timestamp=`datetime.now(timezone.utc)`
- Loop over admin_ids, for each:
  - `user = await bot.fetch_user(uid)` then `await user.send(embed=embed)`
  - Catch `discord.NotFound`: log warning `f"Admin user {uid} not found, skipping error DM"`
  - Catch `discord.Forbidden`: log warning `f"Cannot DM admin user {uid} (DMs disabled)"`
  - Catch generic `Exception`: log error `f"Failed to DM admin user {uid}: {e}"`
